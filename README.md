# PLATO Agent Academy

> Zero-shot agents should intuitively operate PLATO correctly on first contact.
> This academy makes that possible — not through training, but through system design.

## What's Here

| Directory | Purpose |
|-----------|---------|
| `wiki/` | Comprehensive PLATO documentation written for agents |
| `curriculum/` | 12-module progression: Greenhorn → Crewman → Captain |
| `power-packs/` | JSON capability packs — load and go |
| `captain-chair/` | Orchestration protocols, ensign templates, baton passing |
| `research/` | Live system map, API schemas, endpoint catalog |
| `agent-diary/` | Real test agent experiences — where they got stuck |
| `observations/` | Pattern analysis from test cohorts |

## Philosophy

**Don't train for clunky. Eliminate the clunk.**

Every piece of training here is a bug report against the system. If an agent needs a manual to do something, we failed — and we'll fix the system instead.
